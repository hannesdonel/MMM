MMM Documentation

Welcome! If you're here, you want to know how things work here. Lucky you, this documentation will help you with that.

Request URL HTTP Method Request data format Response data format
Get list of all movies /movies GET - A JSON object holding all data about all movies.
Return data about a single movie by title. /movies/[Title] GET - A JSON object holding data about a single movie, containing description, genre, director and image URL. Example:

{
"name": "movieName",
"descripction": "This is the most funniest Mafia movie you'll ever get to see.",
"genre": "Comedy",
"director": "Al Capone",
"imageUrl": "www.link.to/movie/coverartwork"
}
A search query returning data about a genre or author by name. /movies/?genre=[genre] or /movies/?author=[author] GET The search query should contain "genre" or "author". A JSON object holding all movies assigned to a specific genre or author.
Return data about a director. /movies/directors/[Name] GET - A JSON object holding data about a director containing bio, birth year and death year. Example:

{
"bio": "Al Capone was the most funniest movie director ever. Especially when it comes to Mafia "movies": "",
"birth": "1927",
"deathYear": "2027"
}
Allow new users to register. /users POST A JSON object holding data containing first name, last name, age and mail adress. Example:

{
"firstName": "Leonardo",
"lastName": "DiCaprio",
"age": "17",
"mail": "leo@hotmail.com"
} 		A verification message if the registration has been succesfull or not.
Allow users to deregister by ID. /users/:id DELETE The ID of the user to be deleted should be put after /users/ in the URL." A verification message if the deregistration has been succesfull or not.
Allow users to update their user data by ID. /users/:id PUT A JSON object holding data conatining the user data to be changed. Example:

{
"firstName": "New",
"lastName": "Username"
} 		A verification message that a new username was succesfully assigned or failed.
Allow users to add a movie to their list of favorites. /users/:id/favorites PUT A JSON object holding the name of the film to be added. Example:

{
name: "movieName"
} 		A message wether the film has been added succesfully to the favorites or not.
Allow users to remove a movie from their list of favorites. /users/:id/favorites DELETE A JSON object holding the name of the film to be removed. Example:

{
name: "movieName"
} 		A message wether the film has been aremoved succesfully from the favorites or not.
